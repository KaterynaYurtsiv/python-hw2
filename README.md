# Python Homework 2

## Description
This program works with a one-dimensional array of 200 random integers from -100 to 100.

The program provides a Tkinter interface with the following functions:
- create an array
- show the array
- filter positive numbers
- replace negative numbers with zeros
- calculate the average value of the resulting array

## Technologies
- Python
- Tkinter

## How to run
python Task2.py
